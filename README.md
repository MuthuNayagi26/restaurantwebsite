MCA Academic Project - Online Restaurant Website using Full Stack

Key Features:
  Interactive and categorized menu display 
  Add-to-cart functionality with real-time updates 
  Online ordering and checkout system 
  Admin dashboard for managing menu items and customer order details 
  Real-time order storage in database

Technologies I used in my Project: 
  Frontend: HTML, CSS, JavaScript, React.js 
  Backend: Node.js, Express.js 
  Database: MongoDB

How to Run the Project?
  Step 1 -> Open in Integrated Terminal the Folder "server1-hotelreact"
  Step 2 -> Type this Command
            node server.js
            You should see -> Connected to MongoDB 
  Step 3 -> Open in Integrated Terminal the Folder "user1-hotelreact"
  Step 4 -> Type this command
            npm start
            It should navigate to a browser and "My Project(User Side) will be displayed"
            Here, Add some items to Cart and Click "Proceed to Order"
            Please provide your details like Name, Address, Contact No, Add a Note, Delivery Location
            And then Click "Place Order"
            Now a Pop-up will shows "Order Placed Successfully"
  Step 5 -> Back to Vscode, Open in Integrated Terminal the Folder "admin1-hotelreact"
  Step 6 -> Type this command
            npm start
            It should navigate to a browser and "My Project(Admin Side) will be displayed"
            Here provide the Admin Credentials
            Username - admin
            Password - password123
  Step 7 -> Click "Order Detail" Module
            Here we can see the Order Details with complete details which is,
            Name, Address, Contact no, Order Date&Time, Items with Quantity, Special Note, Location, Total Amount 
  Step 8 -> In Addition to that, the Order Details are also added dynamically to the MongoDB Database
